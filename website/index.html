FAQ

Frequently Asked Questions

Is Codiner free? What's the catch?

Yes, Codiner is completely free and open-source. There are no catches or hidden costs. We offer Pro and Max plans for users who want additional features and AI credits, but the core app remains free forever.

What can you build with Codiner?

You can build any kind of web application with Codiner—from simple landing pages to complex full-stack applications. Our Supabase integration allows you to add authentication, databases, and real-time features to your apps.

Can you build desktop apps with Codiner?

Currently, Codiner focuses on web applications. However, you can use the generated web apps as the foundation for desktop applications using tools like Electron or Tauri.
